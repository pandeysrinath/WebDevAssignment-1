Operating System

An operating system usually provides common facilities for programs running on the system to use the systemâ€™s hardware resources.

Things like process support code (threads, locks, spawning and killing processes), storage support code (file creation & manipulation, device support), graphical support code (mode setting, hardware accelerated drawing, vblank sync, memory allocation), other sorts of IO (serial & parallel ports, USB support, keyboard and mouse support), memory allocation and so on.

Some provide very little more than a file system.

It depends on the purpose of the system.
